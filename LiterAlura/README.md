### Descrição do desafio
#### Objetivo

* Construir um catálogo de livros que ofereça interação textual (via console) com os usuários
* 5 opções de interação
* Livros buscados da API Gutendex

#### Tecnologias e bibliotecas utilizadas

- java (JDK 25)
- spring-dotenv 4.0.0 (para carregar variáveis de ambiente)
- Jackson  2.20.1 (desserialização de dados da API)

### Como utilizar
 
 <ol>
    <li>Clone ou faça o download deste repositório;</li>
    <li>Crie um arquivo .env utilizando o arquivo .env.example como base e preencha com seus dados do banco de dados;</li>
    <li>Rode o código da main e siga as orientações do menu no terminal.</li>
 </ol><br/>

 ### Funcionalidades concluidas
 * Menu de interação via console
 * Consulta ao banco de dados
 * Rotas para listar livros, autores, top5 e exibir os detalhes.
